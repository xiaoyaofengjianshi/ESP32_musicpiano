# 项目名称
2025大连理工大学电气创新实践基地极创组考核项目 - 基于ESP32的电子琴

## 项目功能

基于2025电气创新实践基地极创组考核要求，目前ESP32电子琴实现的功能如下：
(1)开屏LED灯组快速闪烁形成彩虹灯效果，同时OLED屏播放猫猫点阵图
(2)OLED屏字模显示
(3)多屏菜单功能
(4)OLED屏边框流动特效
(5)通过A、B两个按键实现OLED屏翻页功能
(6)放音界面按下琴键驱动蜂鸣器播放音符功能
(7)放音界面按下琴键OLED屏提示功能
(8)放音界面按下琴键OLED屏特效
(9)放音界面多琴键同时按下优化功能
(10)放音界面按住E键、L键实现升调、降调功能，OLED屏反馈功能
(11)放音界面按键后的灯组反馈效果
(12)歌曲界面播放多首音乐（小星星、天下相亲与相爱）功能（E键）
(13)播放音乐时LED悦动特效
(14)播放音乐时OLED屏边框动态特效
(15)歌曲播放暂停功能（F键）
(16)亮度调节界面WS2812灯组亮度调节功能，按E键变暗，按L键变亮
(17)音量调节界面无源蜂鸣器音量调节功能，按E键减少，按L键增加
(18)WS2812循环彩虹灯功能
(19)彩蛋猫猫点阵图，无法通过向前翻页访问防止误触

## 快速开始
### 1. 环境搭建
本项目开发环境为visual studio code，需要安装ESP-IDF插件，并配置好相关的环境变量。
特别注意，需要手动添加以下两个相关库：
espressif/led_strip: ^2.5.3
espressif/ssd1306: '*'

### 2. 编译烧录
分别进行编译和烧录即可运行。
