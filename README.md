# 2025极创组招新考核

## ESP32驱动的电子琴

要求使用提供的ESP32板卡，使用ESP-IDF完成本次考核。

### 基本要求：

- 使用PWM驱动无源蜂鸣器发声
- 驱动电容触摸芯片，识别不同按键，发出不同音调声音
- 使用SPI驱动0.96'OLED显示屏，显示当前按下了什么按键
- 驱动12颗WS2812B，发出不同颜色的光
- 代码规范化，使用图体现项目结构

### 发挥部分：

- 优化显示屏显示效果，例如UI设计、音频响应
- 优化WS2812响应效果，设计LED动画响应按键
- 优化蜂鸣器驱动效果，例如实现踏板、和声等效果
- 使用蓝牙或者WIFI功能，设计远程控制方案
